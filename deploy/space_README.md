# Explainable Fraud Detection — live demo

Real-time credit-card fraud scoring with a SHAP explanation of the top contributing
features. LightGBM + (optional) LSTM under a decision-level fusion design.

Paste a transaction → probability, verdict (Normal / Fraud / Expert-Checking), and the
feature contributions behind it.

**Code & write-up:** https://github.com/Leonardasvekrikas-source/fraud-detection-system
